The boundary

Native memory pressure, contained.

The hostile side is allocator pressure and high-velocity token callbacks. The Crashless Boundary admits only safe loads, backpressures bursts, and emits a steady stream to Avalonia.

System stability primitives

Built for desktop apps that cannot afford to disappear.

Predictive Gating

CrashlessLLM estimates model bytes, worst-case KV cache, and margin before native allocation. Unsafe loads become explicit diagnostics, not surprise process death.

Zero UI Blocking

Generation runs on a native worker and streams through IAsyncEnumerable<string>. When the UI slows down, backpressure lands on the worker, not the window.

GC-Safe Streaming

Ephemeral UTF-8 pointers are copied immediately, callback delegates stay pinned for the session lifetime, and SafeHandle-owned native resources release deterministically.

Sampling Controls

Temperature, top-k, top-p, min-p, repeat penalty, repeat window, seed, and max tokens are configured from C# and translated into a native sampler chain.

Chat Templates

Format conversations for Llama 3, ChatML, Mistral, Gemma, and Phi without scattering prompt syntax across your Avalonia view models.

Accurate KV Introspection

After a safe load, query model layers, heads, context training size, and exact KV bytes per token for diagnostics and memory-aware UI reporting.

GPU Backend Detection

Ask the native library which backends were compiled in — Metal, CUDA, Vulkan, ROCm, or SYCL — before presenting acceleration options.

Native Release Pipeline

Cross-platform CI builds macOS, Linux, and Windows native artifacts. The NuGet publish script builds, tests, packs, verifies, and pushes.

Why developers switch

Naive wrappers optimize for access. CrashlessLLM optimizes for survival.

Naive llama.cpp wrapper

  • Loads first, discovers OOM later
  • Pushes tokens faster than UI can paint
  • Lets C++ lifetime leak into managed code
  • Hardcodes sampling, token limits, and prompt formats

CrashlessLLM

  • Rejects unsafe allocations before load
  • Bounds token pressure with channel backpressure
  • Owns native lifetime through SafeHandle
  • Exposes sampling, chat templates, GPU flags, and model architecture
